# LancerChat
Fully working command line based LAN only chat intended for use within the Lancer ttrpg. Still in active development.

Current objectives to modify in order of priority.
- Not fully instanceable. Annoying if you wish to keep various servers.
- Not fully thread safe (But causing very minor issues)
- Lack of personalization
- Self made command line style UI. Instead of cmd a better looking one.
- Add non-LAN usability
- Comp/Con compatibility (Unlikely)
